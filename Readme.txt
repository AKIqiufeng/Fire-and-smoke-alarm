本项目基于普中51开发板A2，以0.96 OLED屏作为显示模块
本程序主要为3个模式，，开机自动进入菜单，可选择温湿度检测模式，烟雾报警模式，火光报警模式
项目为自学时练手的项目，可能有各种问题
使用前请先检查并修改各端口号为自己开发板对应的。
 测试使用开发板为普中51开发板A2
I2C端口：SCL:P02
	SDA:P01
独立按键：K1:P31
	K2:P30
	K3:P32
	K4:P33
DHT11:P00
火焰传感器模块:P03
烟雾传感器模块:P04
无源蜂鸣器:P25
(如需使用有源蜂鸣器，
替换Beep.c与Beep.h为工程文件夹中有源蜂鸣器文件夹中的模块
并删除定时器0中断函数中的Beep_Loop();语句
有源蜂鸣器端口P07)
